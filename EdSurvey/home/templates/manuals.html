Личности и Роли по-умолчанию

Пользователь "Сайта"

  • Регистрируется самостоятельно
  • При создании автоматически получает "Корневую Личность", которая:
    • принадлежит "Корневой Организации" "Сайта"
    • наделяется "Ролями":
      • "Тестируемый" (Testee)
      • "Владелец контента" (Owner)
  • Может быть лишена "Ролей" "Модератором" администрацией "Сайта" (см. "Матрицу прав")

Пользователь "Сообщества"

  • Регистрируется по "Приглашению"
  • При создании автоматически получает "Корневую Личность", которая:
    • принадлежит "Корневой Организации" "Сообщества"
    • наделяется "Ролями":
      • "Тестируемый" (Testee)
      • "Владелец контента" (Owner)
  • Может быть лишена "Ролей" "Модератором" администрацией "Сообщества" (см. "Матрицу прав")

Пользователь "Корпорации"

  • Регистрируется либо пакетом либо "Личностью" "Организации" "Клиента" с "Ролью" "Менеджер" (Account Manager)
  • При создании автоматически получает "Корневую Личность", которая:
    • принадлежит "Корневой Организации" "Корпорации"
    • "Ролями" НЕ наделяется
  • Всеми правами Пользователя управляет персонал "Корпорации"
